Настройка Ubuntu сервера

После установки Ubuntu рекомендуется произвести обновление пакетов и самой системы. В терминале вводим следующие команды:

sudo apt update -y sudo apt upgrade -y

При перовом использовании команды sudo система попросит ввести пароль администратора.
