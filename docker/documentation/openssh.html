Установка OpenSHH

После обновления системы установим OpenSSH, который позволит удаленно пользоваться системой. Для этого в терминале выполним команду:

sudo apt install openssh-server -y

После установки OpenSSH, перезапустим систему командой:

sudo reboot

Дальнейшие манипуляции с сервером будут выполняться через ssh
